COMP 3104 – DevOps

Course Overview:
COMP 3104 is a course focused on DevOps principles, tools, and practices to streamline software development and deployment. The course emphasizes automation, continuous integration/continuous deployment (CI/CD), infrastructure as code (IaC), and collaboration between development and operations teams.

Topics Covered:
Introduction to DevOps – Understanding the culture, principles, and benefits of DevOps.
Version Control with Git & GitHub – Managing code repositories, branching strategies, and collaboration.
Continuous Integration (CI) & Continuous Deployment (CD) – Setting up automated build and deployment pipelines using Travis CI, Jenkins, or GitHub Actions.
